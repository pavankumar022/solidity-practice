# Solidity Projects

A collection of Solidity smart contracts built while learning Ethereum blockchain development. This repository contains hands-on projects that demonstrate fundamental and intermediate Solidity concepts using Remix IDE.

## Tech Stack

- Solidity
- Ethereum
- Remix IDE

```
└── ...
```

## Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/<your-username>/solidity-projects.git
```

Or download the repository as a ZIP file.

### 2. Open Remix IDE

Go to:

https://remix.ethereum.org/

### 3. Create a Solidity File

- Open the **File Explorer**.
- Inside the **contracts** folder, create a new file.
- Give the file a **`.sol`** extension.
- Copy the code from any project in this repository and paste it into the file.

### 4. Compile

- Open the **Solidity Compiler** tab.
- Select the appropriate compiler version.
- Click **Compile**.

### 5. Deploy

- Open **Deploy & Run Transactions**.
- Select **Remix VM** as the environment.
- Click **Deploy**.
- Interact with the deployed contract.
